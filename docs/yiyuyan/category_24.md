## 📌 函数: HDNT_窗口句柄取PID
### 描述
```

```
### 原型
```
__int64 __stdcall HCNT_GetWindowProcessId(__int64 hwnd);
```
### 参数
```
__int64 hwnd - 窗口句柄
```
### 返回值

# 🛰️ 谷歌内置浏览器模块[GB]
## 📌 函数: HCGB_Attach1
### 描述
```
附加内置浏览器，附加监听模式1
```
### 原型
```cpp
__int64 __stdcall HCGB_Attach1(__int32 窗口序号);
```
### 参数
```
__int32 窗口序号 - 窗口序号(从1开始)
```
### 返回值
```
查看返回值表
```
### 注意事项
```
必须游戏进程自带内置谷歌浏览器,才会附加模块成功
由于每个游戏不同,需要定制浏览器接口
当前默认的内置谷歌浏览器接口仅适合夜鸦
```

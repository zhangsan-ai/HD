## 📌 函数: HD资源.设置路径Ex
### 描述
```
设置资源路径
```
### 原型
```
void __stdcall  HCRES_SetResPathEx(char *path, __int32 windIndex = -1); 
```
### 参数
```
__int32 windIndex - 窗口序号 -1表示全局资源(同 HCRES_SetResPathA/W 一个效果) 大于等于0表示设置的是窗口全局资源路径
char* path - 路径
```
